Versioning definition 

 [Major].[Minor].[Revision].[Build]
 
 EG => 1.0.100.16
 
 [Major] => Major number.
 [Minor] => Minor number.
 [Revision] => Days since the date: 01.01.2022.
 [Build] => How many times built that day.

 1.0.100.16 
	=> equals version 1.0

	=> built on 11.04.2022
	=> built 16 times that day.
